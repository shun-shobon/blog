# Next.js template

## Usage

### Install

```shell
$ pnpm install
```

### Build

```shell
$ pnpm build
```

### Start production server

```shell
$ pnpm start
```

### Start development server

```shell
$ pnpm dev
```

### Start storybook server

```shell
$ pnpm storybook
```

### Run type checker

```shell
$ pnpm type
```

### Run linter

```shell
$ pnpm lint
```

### Run formatter

```shell
$ pnpm format
```

### Run test

```shell
$ pnpm test
```
